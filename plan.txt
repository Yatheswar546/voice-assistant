Development Plan

Rather than trying to build everything at once, I'd recommend this order:

Phase 1: Static UI
Create the Next.js project structure
Set up Tailwind CSS and shadcn/ui
Build the dark layout
Create the sidebar
Create the header
Create chat bubbles
Build the bottom input
Make the UI responsive

Phase 2: Voice UI
Add microphone button states
Add listening animation
Add speaking animation
Add typing indicator

Phase 3: Backend Integration
Connect the chat to your AI backend
Add streaming responses
Connect speech-to-text
Connect text-to-speech

Phase 4: Production Features
Conversation history
Authentication
Database
Settings
Deploy to production
